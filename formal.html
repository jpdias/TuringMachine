Formal Definition

A Turing machine is formally defined (one-tape) as a 7-tuple M = ( Q, Γ, b, ∑, δ, q0, F ) where:

  • Q is a finite, non-empty set of states
  • Γ is a finite, non-empty set of the tape alphabet/symbols
  • b ∈ Γ is the blank symbol (the only symbol allowed to occur on the tape infinitely often at any step during the computation)
  • ∑ ⊆ Γ ∖ {b} is the set of input symbols
  • q0 ∈ Q is the initial state
  • F ⊆ Q is the set of final or accepting states.
  • δ: Q ∖ F × Γ → Q × Γ × {L,R} is a partial function called the transition function, where L is left shift, R is right shift.

Anything that operates according to these specifications is a Turing machine.

Hopcroft and Ullman (1979, p. 148)
